commands : 
npx playwright codegen : to start local.
npx playwright test : run the test
npx playwright show-report : shows report   
npx playwright test --ui : u.I test
Add // @ts-check at the start of each test file when using JavaScript in VS Code to get automatic type checking.
npx playwright test --headed : for live GUI
npx playwright test --debug : to debug the test.
